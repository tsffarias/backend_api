# Backend API Starting Point

This is a Backend API Starting Point for projects: Node.js + Express + TypeScript + Mongoose and some other packages.

### Version: 1.0.0

### Usage

```sh
$ npm install
```

```sh
# run api dev mode
$ npm run dev
# build project
$ npm run build
# run api from /dist
$ npm run start

# Visit http://localhost:3000
```
